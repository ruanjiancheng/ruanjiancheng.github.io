Vue

什么是 Vue

  • Vue.js（读音 /vjuː/, 类似于 view） 是一套构建用户界面的渐进式框架。

  • Vue 只关注视图层， 采用自底向上增量开发的设计。

  • Vue 的目标是通过尽可能简单的 API 实现响应的数据绑定和组合的视图组件。

  1. https://unpkg.com/vue/dist/vue.js
  2. https://cdnjs.cloudflare.com/ajax/libs/vue/2.1.8/vue.min.js
